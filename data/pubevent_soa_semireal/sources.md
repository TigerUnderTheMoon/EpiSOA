# PubEvent-SOA Semi-real Sources

This directory is a semi-real experiment template built from public-web style snippets. The included examples use `example.org` URLs and fictionalized public-event content so the repository remains self-contained and safe for tests.

For real experiments, only add publicly accessible pages that do not require login, paywall bypass, private groups, or anti-bot circumvention. Keep the source page URL, platform name, timestamp, and short text snippet needed for attribution. Do not store user profile URLs or direct personal identifiers.

## Source Types

- `news`: public news or civic reporting pages.
- `official`: official public response, agency update, or public notice.
- `forum`: publicly accessible community or organization forum posts.
- `blog`: public organization or community blog posts.
- `other`: public web pages that do not fit the categories above.
