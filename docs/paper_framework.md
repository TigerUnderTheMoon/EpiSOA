# Paper Framework

## Introduction
Motivate Evidence-grounded Stakeholder Opinion Attribution for public events and explain why attribution must be grounded in traceable evidence.

## Related Work
Position the work relative to opinion mining, stance detection, retrieval-augmented generation, event-centric reasoning, and evidence verification.

## Task Definition
Define the input evidence corpus, public event description, time window, and structured attribution output schema.

## Dataset
Describe the event queries, evidence records, gold attribution tuples, gold event chains, and annotation protocol.

## Method
Present the EpiSOA pipeline: C-FSM evidence collection, diversity-aware retrieval, stakeholder-centered graph building, event-chain retrieval, schema-constrained reasoning, and evidence verification.

## Experiments
Report the main experiment, baselines, ablations, metrics, and reproducibility settings.

## Discussion
Analyze strengths, limitations, error patterns, case studies, and implications for evidence-grounded attribution.

## Conclusion
Summarize the contribution and outline future work.
